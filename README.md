# Project

## Tech Tut: draw.io (diagrams.net) — Business Diagramming Tutorial

This repository contains the Tech Tut tutorial for **draw.io** (diagrams.net), created for BIT 4454: Business Analysis Seminar in IT.

### Contents

- **[TechTut-DrawIO-Tutorial.md](TechTut-DrawIO-Tutorial.md)** — The comprehensive Word document tutorial (in Markdown format) covering:
  - Part A1: Technology area description (what draw.io does, why it's needed, who uses it)
  - Part A2: Competitor comparison table (draw.io vs. Visio, Lucidchart, Miro, Creately, Gliffy)
  - Part B: Step-by-step tutorial with a novel business application (Summit Outdoor Gear Co. order fulfillment system), including:
    - Organization charts
    - BPMN process diagrams
    - System architecture diagrams
    - Database ER diagrams
    - Customer journey maps
    - Network infrastructure diagrams
    - Collaboration, exporting, and sharing
    - Advanced features and tips
  - Part 4: Best additional tutorial materials (online tutorials, books, resources)

### Novel Business Application

The tutorial uses a fictional company, **Summit Outdoor Gear Co.**, as a novel example case. This mid-size e-commerce retailer specializing in outdoor equipment needs to document and improve its order fulfillment process. The tutorial demonstrates how to use draw.io to create six different types of business diagrams for this scenario.
